Виджет "ElemRoutesNavigator" - Навигация по страницам
═══════════════════════════════════════════════════════════════════════════════

ОПИСАНИЕ
────────────────────────────────────────────────────────────────────────────────

Виджет ElemRoutesNavigator предназначен для создания навигации по страницам
приложения. Он автоматически получает список страниц из раздела "Ссылки" и
отображает их в виде меню навигации.

ВАЖНО: К данному виджету НЕ НУЖНО подключать датасет. Все данные о страницах
(названия и пути) автоматически берутся из раздела "Ссылки" в редакторе
приложения.

[МЕСТО ДЛЯ СКРИНШОТА: Виджет навигации на странице]

ПОДКЛЮЧЕНИЕ СТРАНИЦ К ВИДЖЕТУ
═══════════════════════════════════════════════════════════════════════════════

Виджет автоматически отображает все страницы, для которых созданы ссылки в
разделе "Ссылки".

КАК СОЗДАТЬ ССЫЛКУ НА НОВУЮ СТРАНИЦУ
────────────────────────────────────────────────────────────────────────────────

Чтобы страница появилась в навигационном виджете, необходимо создать для неё
ссылку:

1. Перейдите в подраздел "Ссылки"
2. Нажмите на кнопку "Новая ссылка" для вызова окна редактора ссылки

[МЕСТО ДЛЯ СКРИНШОТА: Раздел "Ссылки" с кнопкой "Новая ссылка"]

В редакторе ссылки заполните следующие поля:

• Заголовок - название страницы, отображаемое на вкладке браузера и в списке
  ссылок
• Название - название, которое будет отображаться в виджете навигации
• Ссылка - путь для перехода на страницу в формате /<название ссылки>
  (например: /about, /contacts)
• Страница - выберите из выпадающего списка страницу, на которую будет вести
  ссылка

[МЕСТО ДЛЯ СКРИНШОТА: Редактор ссылки с заполненными полями]

После заполнения всех полей нажмите кнопку "Сохранить".

После сохранения:
• В подразделе "Ссылки" появится новая ссылка
• В подразделе "Страницы" у созданной страницы, ниже названия, появится
  название ссылки
• Страница автоматически появится в виджете навигации

[МЕСТО ДЛЯ СКРИНШОТА: Созданная ссылка в списке]

Сохраните проект после внесения изменений.

НАСТРОЙКА ВИДЖЕТА
═══════════════════════════════════════════════════════════════════════════════

ПАНЕЛЬ "НАСТРОЙКИ ВИДЖЕТА"
────────────────────────────────────────────────────────────────────────────────

[МЕСТО ДЛЯ СКРИНШОТА: Панель "Настройки виджета"]

Основные параметры
──────────────────

• Показывать заголовок - включает отображение заголовка над навигацией
• Заголовок - текст заголовка (по умолчанию: "Навигация")
• Показывать ссылки страниц - показывает путь ссылки рядом с названием страницы
• Активная страница - подсвечивает текущую открытую страницу в навигации

Ориентация навигации
────────────────────

Выберите один из вариантов расположения навигации:

• Вертикальная - кнопки расположены вертикально друг под другом
• Горизонтальная - кнопки расположены горизонтально в ряд
• Выпадающий список - навигация скрыта в выпадающем меню
• Компактное меню - навигация открывается по клику на иконку меню

[МЕСТО ДЛЯ СКРИНШОТА: Примеры разных типов ориентации навигации]

Для вертикальной ориентации, выпадающего списка и компактного меню доступна
настройка "Позиционирование кнопок" (по левому краю, по центру, по правому краю).

Дополнительные настройки
────────────────────────

• Текст меню (для выпадающего списка) - текст, отображаемый на кнопке
  выпадающего списка


ПАНЕЛЬ "НАСТРОЙКИ ПАГИНАЦИИ"
────────────────────────────────────────────────────────────────────────────────

[МЕСТО ДЛЯ СКРИНШОТА: Панель "Настройки пагинации"]

Эта панель позволяет настроить внешний вид и поведение навигации.

Пагинация
─────────

• Включить пагинацию - ограничивает количество отображаемых элементов навигации
• Тип пагинации:
  - Прокрутка - добавляет вертикальную прокрутку при большом количестве страниц
  - Нумерация страниц - добавляет кнопки с номерами страниц для переключения
• Элементов на странице - количество страниц, отображаемых одновременно
  (по умолчанию: 6)
• Цвет активной страницы - цвет выделения текущей страницы в нумерации
  (только для типа "Нумерация страниц")

[МЕСТО ДЛЯ СКРИНШОТА: Пример нумерации страниц в меню]

Режим открытия меню
───────────────────

Для выпадающего списка и компактного меню:

• По клику - меню открывается при клике на кнопку
• При наведении - меню открывается при наведении курсора

Стилизация
──────────

• Стиль кнопок:
  - Заполненные - кнопки с фоновым цветом
  - С обводкой - кнопки только с рамкой
  - Текстовые - кнопки без фона и рамки

• Цвета:
  - Цвет активной кнопки
  - Цвет при наведении
  - Фон контейнера
  - Цвет текста

• Размеры:
  - Размер шрифта
  - Отступ кнопок
  - Расстояние между кнопками
  - Скругление углов

• Семейство шрифта - позволяет изменить шрифт навигации

Тени
────

• Тень контейнера - добавляет тень всему контейнеру навигации
  (CSS box-shadow, например: 0 4px 6px rgba(0,0,0,0.1))
• Тень кнопок - добавляет тень кнопкам навигации
• Тень выпадающих меню - добавляет тень выпадающим меню
  (для dropdown и компактного меню)

Границы (только для выпадающего списка и компактного меню)
──────────────────────────────────────────────────────────

• Показывать границу списка - добавляет рамку вокруг выпадающего меню
• Цвет границы списка - цвет рамки меню
• Показывать границу кнопки - добавляет рамку вокруг кнопки открытия меню
• Цвет границы кнопки - цвет рамки кнопки


ПАНЕЛЬ "РЕЖИМ ЭКСПЕРТА"
────────────────────────────────────────────────────────────────────────────────

[МЕСТО ДЛЯ СКРИНШОТА: Панель "Режим эксперта"]

Панель "Режим эксперта" позволяет создавать иерархическую навигацию с разделами
и подразделами.

Включение иерархии
──────────────────

• Включить иерархию - активирует режим вложенной навигации
• Переходить при клике на раздел - при клике на раздел с подстраницами
  происходит переход на страницу раздела и разворачиваются подстраницы

Управление иерархией и порядком
────────────────────────────────

В режиме эксперта отображается визуальный редактор иерархии со списком всех
страниц.

[МЕСТО ДЛЯ СКРИНШОТА: Визуальный редактор иерархии с drag & drop]

Создание иерархии:
• Перетащите страницу НА элемент (в центр) - страница станет подстраницей
  выбранного раздела
• Перетащите страницу ДО/ПОСЛЕ элемента (к краям) - изменится порядок страниц

Визуальные подсказки:
• Серый фон - обычная страница без подстраниц
• Зелёный фон - раздел (страница 0 уровня), от которой строится иерархия
  (имеет подстраницы)
• Синий фон - подстраница (вложенная в раздел)
• Отступ слева показывает уровень вложенности

[МЕСТО ДЛЯ СКРИНШОТА: Пример визуальной подсветки иерархии
(зелёный - родитель, синий - дети)]

Действия с страницами:
• ✓ - страница включена и отображается в навигации
• 👁️ - страница отключена и скрыта из навигации (нажмите чтобы включить)
• ✕ - убрать все подстраницы из раздела (превратить подстраницы в обычные
  страницы)
• ↑ - вынести подстраницу на верхний уровень (сделать обычной страницей)


ПАНЕЛЬ "Я ДИЗАЙНЕР"
────────────────────────────────────────────────────────────────────────────────

[МЕСТО ДЛЯ СКРИНШОТА: Панель "Я дизайнер"]

Панель для продвинутой CSS-стилизации отдельных элементов навигации.

Доступна стилизация следующих элементов:

• Основной контейнер - контейнер всей навигации
• Заголовок навигации - заголовок над меню
• Кнопка страницы - обычное состояние кнопки
• Кнопка при наведении - состояние при наведении курсора
• Активная кнопка - текущая открытая страница
• Кнопка выпадающего списка - кнопка открытия dropdown меню
• Меню выпадающего списка - выпадающее меню
• Кнопка компактного меню - кнопка открытия компактного меню
• Компактное меню - выпадающее меню компактного режима
• Линии иконки меню (дефолт) - стили дефолтной иконки меню
  (три горизонтальные линии)
• Текст ссылки страницы - стиль отображения пути ссылки
• Стили иконок иерархии - стили иконок развернуть/свернуть для разделов

[МЕСТО ДЛЯ СКРИНШОТА: Редактор CSS для элемента с кнопкой "Заполнить текущими"]

Для каждого элемента можно:
1. Увидеть текущие настройки из других панелей
2. Ввести собственные CSS-свойства
3. Автоматически заполнить поле текущими настройками кнопкой
   "⌨️ Заполнить текущими"
4. Сбросить стили для конкретного элемента

Кнопка "Сбросить все стили и иконки" удаляет все пользовательские стили и
иконки, возвращая виджет к настройкам по умолчанию.

Настройка иконок компактного меню
──────────────────────────────────

[МЕСТО ДЛЯ СКРИНШОТА: Секция настройки иконок компактного меню]

Позволяет заменить стандартные три линии (☰) на любые иконки из библиотеки
Material Design Icons (https://materialdesignicons.com/):

• Иконка закрытого меню - иконка, отображаемая когда меню закрыто
  (например: mdi-menu)
• Иконка открытого меню - иконка, отображаемая когда меню открыто
  (например: mdi-close)
• Размер иконки (font-size) - размер иконки (по умолчанию: 1.5rem)

Настройка иконок иерархии
──────────────────────────

[МЕСТО ДЛЯ СКРИНШОТА: Секция настройки иконок иерархии]

Позволяет заменить стандартные символы (▶/▼) на иконки для разделов с
подстраницами:

• Иконка развернутого раздела - иконка когда раздел раскрыт
  (например: mdi-chevron-down)
• Иконка свернутого раздела - иконка когда раздел свёрнут
  (например: mdi-chevron-right)
• Размер иконки (font-size) - размер иконки (по умолчанию: 0.75rem)

[МЕСТО ДЛЯ СКРИНШОТА: Пример результата с кастомными иконками в навигации]

ПРИМЕЧАНИЯ
═══════════════════════════════════════════════════════════════════════════════

• Приложение всегда запускается с первой ссылки "/"

• Если в режиме запуска проекта нужно перейти на другую страницу, а на главной
  странице еще нет виджетов для перехода, можно скопировать название ссылки
  (все после значка "/") под названием страницы в подразделе "Страницы" и
  добавить его к адресу в браузере

• При создании иерархии в режиме эксперта, изменения применяются только к
  отображению в навигации и не влияют на структуру ссылок в редакторе

• Все изменения стилей применяются в реальном времени в режиме редактора
